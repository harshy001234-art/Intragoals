# Demo Credentials

## Judge-friendly demo path

The fastest way to review Intragoals is the built-in demo workspace. Use the role switcher in the authenticated UI to move between the employee, manager, and admin experiences without needing seed accounts.

Demo personas:

- Employee: Neha Sharma
- Manager: Priya Iyer
- Admin: Rohan Kapoor

## If Supabase auth is enabled

- Register a workspace account from `/register`, or
- Sign in with an existing workspace user from `/login`

The first account for a new organization domain is bootstrapped as the admin for that workspace.
